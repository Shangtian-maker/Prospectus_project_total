# 招股书结构化抽取系统  
## IPO Prospectus Structured Extraction System

基于 PDF 解析、章节定位、Rule+LLM 混合抽取与数据校验的 IPO 招股书股权事件结构化分析系统。

## 项目简介

本项目面向 IPO 招股书中的增资、股权转让、股本变化等信息，实现从非结构化 PDF 到结构化事件数据的自动抽取，并通过证据回溯和一致性校验提高结果可靠性。

## 技术流程

PDF解析  
→ 章节定位  
→ 事件切块  
→ Rule + LLM抽取  
→ Schema校验  
→ Cross-check验证

## 四周工作总结

**Week 1：** 完成 PDF 解析与抽取原型，实现基础事件识别和证据记录。

**Week 2：** 建立统一数据结构、Gold标准和校验机制，提高结果可验证性。

**Week 3：** 引入 Rule+LLM 混合抽取模式，优化事件识别和结构化输出。

**Week 4：** 优化章节定位模块，减少全文输入噪声，实现精准章节抽取。

## 当前成果

- PDF自动解析
- 招股书章节定位
- 股权事件抽取
- JSON结构化输出
- Gold标准评价
- 数据一致性校验

## 技术栈

Python | PyMuPDF | Pandas | Pydantic | LLM

构建一个可解释、可验证、可扩展的 IPO 招股书结构化抽取系统。
